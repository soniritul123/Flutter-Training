Database: collection of Data

there are mainly 3 types of database:

1) Offline Database
2) Online Database
3) Cloud Database

Offline Database: which is simple to use and which is use database as a device storage
            We can use only in mobile application

Online Database: which is use to implement by application
            database hosted on another server and access data using of api

Cloud Database: Firebase
